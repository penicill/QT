# Qt Odom Viewer 更新日志

## V1.1.0 - 2024
### 重构完成 ✓
- **UI界面重构**: 从纯代码创建UI改为使用Qt Designer .ui文件
- **代码结构优化**: 将UI代码与业务逻辑分离，提高可维护性
- **模块化设计**: 创建了独立的TrajectoryFrame和DashboardFrame类
- **Qt5.15.3兼容性**: 优化了代码以确保在Qt 5.15.3版本下稳定运行
- **信号槽兼容性**: 使用Qt5兼容连接语法

### 功能保持 ✓
- [x] 实时显示里程计数据（位置/姿态/速度）
- [x] 美观的偏航角仪表盘显示
- [x] 轨迹图实时绘制与缩放
- [x] 鼠标滚轮缩放轨迹图
- [x] 鼠标左键拖动轨迹图
- [x] 暂停/继续数据更新
- [x] 清除轨迹并重置视图
- [x] 度/弧度单位切换
- [x] 最大500点轨迹记忆
- [x] ROS2节点状态显示

### 新增文件 ✓
- [x] `ui/odom_viewer_window.ui` - Qt Designer UI文件
- [x] `src/ui_odom_viewer_window.h` - UI自动生成的头文件
- [x] `include/qt_odom_viewer/odom_viewer_window_ui.h` - 新窗口类定义
- [x] `src/odom_viewer_window_ui.cpp` - 新窗口实现

### 优化内容 ✓
- [x] 分离了轨迹图和仪表盘为独立组件
- [x] 优化了Qt 5.15.3兼容性
- [x] 改进了代码注释和文档
- [x] 增强了类型安全性
- [x] 标准化了信号槽连接
